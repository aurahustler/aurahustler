### 📂 Featured Projects

#### 🎮 [Minecraft VPS Manager Bot](https://github.com/aurahustler/minecraft-server-bot)
An automated Telegram Bot designed to manage a Minecraft server network running inside `screen` sessions on a remote Linux VPS. 
*   **Key Features:** Status checker (polls players/latency), automatic lifecycle daemon (turns off server if inactive for 5+ minutes to save resources), webhook server on `aiogram` & `aiohttp`, multi-threaded execution to prevent blocking operations.
*   **Stack:** Python, Aiogram, Redis, JavaServer, Screen, Linux.

#### 📈 [Потужнометр (Potuzhnometr)](https://github.com/aurahustler/potuzhnometr)
A real-time high-load web service that tracks geopolitical tension by scraping news from selected Telegram channels and analyzing them using LLMs.
*   **Key Features:** Worker parses posts via MTProto and gets structural JSON analyses from Gemini AI. High-performance Go backend serves client SSE streams directly from Redis. Fast database writes to MariaDB, and Minecraft-style log compression on startup.
*   **Stack:** Go (Fiber), Python, Redis (Pub/Sub), MariaDB, Gemini AI, Docker.
